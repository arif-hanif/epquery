IDF¶

class epquery.IDF(idf_path=None, idf_obj=None)¶

This class is used to parse and handle IDF files.

add_object(obj, index=None)¶

Adds EnergyPlus object to the IDF. If index is None, the object is appended to the end of IDF. Otherwise it’s inserted at index (using list.insert()).

Parameters:obj (list(str)) – EnergyPlus object representation
Return type:None
get_object(index)¶

Returns object stored at index.

Parameters:index (int) – Index of idf list
Returns:Object representation
Return type:list(str)
get_objects()¶

Returns parsed objects. All fields are represented as strings, even numeric values.

Returns:List of lists (objects) of strings (fields)
Return type:list(list(str))
set_object(index, obj)¶

Replaces object at index with obj.

Parameters:

  • obj (list(str)) – E+ object representation

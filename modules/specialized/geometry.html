get_surface_centroid(surface_name, surface_type=’BuildingSurface:Detailed’, relative=False)¶

Returns the surface center point (centroid). The centroid can be in absolute or relative coordinates (relative to the first vertex).

Parameters:
  • surface_name (str) – Surface name
  • surface_type (str) – Surface type, default: BuildingSurface:Detailed
  • relative (bool) – If True the centroid is given in relative coordinates
Return type:

numpy.ndarray with 3 elements
